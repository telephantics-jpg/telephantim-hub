Ollama dual minds for ANY browser (telephantim.com)
===================================================

Goal: Mjolnir = llama3.2, Caduceus = hermes3, working on phones/PCs worldwide.

How it works
------------
- Ollama still runs on YOUR PC (the models live there).
- server.py is the dual-mind API (two system prompts + memory).
- Cloudflare Tunnel gives a public HTTPS URL to that API.
- telephantim.com loads api-config.js with that URL so every visitor uses it.

ONE CLICK (PC must stay on)
---------------------------
1. Start Ollama (models: llama3.2 + hermes3)
2. Double-click:  GO_PUBLIC_BRAINS.bat
3. Wait until it prints a trycloudflare.com URL and pushes api-config.js
4. Anyone opens https://telephantim.com/  → Relic duel uses YOUR Ollama

LOCAL only (you alone)
----------------------
OPEN_LOCAL_AI.bat → http://127.0.0.1:8765/

Notes
-----
- Closing GO_PUBLIC_BRAINS or turning the PC off disconnects public brains.
- Tunnel URL changes each run; the bat re-publishes api-config.js each time.
- No "intent files". Characters never mention Ollama/AI in speech.
- Optional cloud (no PC): Render web service + XAI_API_KEY (not Ollama).
